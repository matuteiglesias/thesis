A Convenient Framework for Working with Location Quotient Values

A Convenient Framework for Working with Location Quotient Values​

In this section, we introduce useful variables for describing any observed location quotient value. We will then discuss the empirical distribution of observations in three examples: exports by country and product, patent applications by region and technology class, and business counts by city and industry.

The 2D Space for Log Location Quotients​

Location quotients are defined as:

LQcp=scp/ScSp/SW=scp/SpSc/SWLQ_{cp} = \frac{s_{cp} / S_c}{S_p / S_W} = \frac{s_{cp} / S_p}{S_c / S_W}LQcp​=Sp​/SW​scp​/Sc​​=Sc​/SW​scp​/Sp​​

where scps_{cp}scp​ is the value associated with categories ccc, ppp. The terms Sc=∑pscpS_c = \sum_p s_{cp}Sc​=∑p​scp​, Sp=∑cscpS_p = \sum_c s_{cp}Sp​=∑c​scp​, and SW=∑cpscpS_W = \sum_{cp} s_{cp}SW​=∑cp​scp​ represent the sizes of entities ccc, ppp, and the table total, respectively. This shows that LQLQLQ is the ratio between the relative size of element scps_{cp}scp​ on column ppp and the relative size of row ccc in the matrix total.

When LQcp=1LQ_{cp} = 1LQcp​=1, we have:

s^cpSW=ScSWSpSW;if LQ^cp=1\frac{\hat s_{cp}}{S_W} = \frac{S_c}{S_W} \frac{S_p}{S_W}; \hspace{1cm} \text{if}\ \hat{LQ}_{cp} = 1 SW​s^cp​​=SW​Sc​​SW​Sp​​;if LQ^​cp​=1

Many studies classify observations by whether they are above or below the LQ=1LQ = 1LQ=1 threshold, transforming LQ levels into a binary variable LQ>1LQ > 1LQ>1.

The terms in the definition can be rearranged as:

LQcp=scp/(ScSpSW)LQ_{cp} = s_{cp} / \left(\frac{S_c S_p}{S_W}\right)LQcp​=scp​/(SW​Sc​Sp​​)

Taking the logarithm of this definition converts the ratio to a difference:

log⁡(LQcp)=log⁡(scp)−log⁡(ScSp/SW)log⁡(LQcp)=log⁡(scp)−[log⁡(Sp)+log⁡(Sc)−log⁡(SW)]y1=x1−x2\begin{split} \log(LQ_{cp}) &= \log(s_{cp}) - \log\left( {S_c S_p}/{S_W} \right)\\ \log(LQ_{cp}) &= \log(s_{cp}) - \left[ \log(S_p) + \log(S_c) - \log(S_W) \right]\\ y_1 &= x_1 - x_2 \end{split}log(LQcp​)log(LQcp​)y1​​=log(scp​)−log(Sc​Sp​/SW​)=log(scp​)−[log(Sp​)+log(Sc​)−log(SW​)]=x1​−x2​​

Here, y1:=log⁡(LQcp)y_1 := \log(LQ_{cp})y1​:=log(LQcp​) is determined by two magnitudes: x1:=log⁡(scp)x_1 := \log(s_{cp})x1​:=log(scp​) (observation) and x2:=log⁡(ScSp/SW)x_2 := \log\left( {S_c S_p}/{S_W} \right)x2​:=log(Sc​Sp​/SW​) (uncorrelated margins expectation).

The definition of LQLQLQ can be seen as a constraint involving three unknowns, lying on a two-dimensional plane. A location quotient observation is fully determined if we have two of the following: y1=log⁡(LQ)y_1 = \log(LQ)y1​=log(LQ), x1=log⁡(scp)x_1 = \log(s_{cp})x1​=log(scp​), or x2=log⁡(ScSp/SW)x_2 = \log\left( {S_c S_p}/{S_W} \right)x2​=log(Sc​Sp​/SW​).

Another convenient variable is y2:=(x1+x2)/2y_2 := (x_1 + x_2) / 2y2​:=(x1​+x2​)/2, the mean between the log observed value x1=log⁡(scp)x_1 = \log(s_{cp})x1​=log(scp​) and the log independent margins expectation x2x_2x2​. This is called the size factor. While the location quotient captures the difference between x1x_1x1​ and x2x_2x2​, the size factor captures the value around which these are situated.

Any pair in x1x_1x1​, x2x_2x2​, y1y_1y1​, y2y_2y2​ is enough to define an LQ observation. These variables are linearly independent among themselves. LQ and size factor capture the asymmetric and symmetric parts of the relation between the observation x1x_1x1​ and the expectation x2x_2x2​. At the threshold LQ=1LQ = 1LQ=1, the size factor y2y_2y2​ matches the value that the observation and the expectation (x1x_1x1​, x2x_2x2​) are taking.

We can describe an LQ observation using any pair of variables. In this paper, both coordinates (y1y_1y1​, y2y_2y2​) and (x1x_1x1​, x2x_2x2​) are used. In the first case, LQ>1LQ > 1LQ>1 values are in the (green) upper hemisphere. In the second case, LQ>1LQ > 1LQ>1 appear in the (green) hemisphere above the identity line, where x1>x2x_1 > x_2x1​>x2​.

This framework is useful for modeling jumps of LQLQLQ over a threshold and dealing formally with issues involving location quotients. It helps analyze sparseness in the data, focusing on entity sizes (ScS_cSc​, SpS_pSp​, SwS_wSw​) reflected in x2x_2x2​. The framework can be generalized to separate entity sizes: x2=log⁡(Sc)+log⁡(Sp)−log⁡(Sw)=xc+xp−xwx_2 = \log(S_c) + \log(S_p) - \log(S_w) = x_c + x_p - x_wx2​=log(Sc​)+log(Sp​)−log(Sw​)=xc​+xp​−xw​. It complements discussions on location quotients with entity sizes and absolute levels x1x_1x1​, as seen in works by Kemeny and Fracasso.

In summary, this framework is robust and convenient for formalizing questions involving location quotients.
