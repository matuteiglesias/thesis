Mean and Variance of Transformed Random Variables​

To estimate the moments (expected value and variance) of the logarithm of aggregate sales, we can utilize a Taylor expansion for the moments of the function f(x):=log⁡(x)f(x) := \log(x)f(x):=log(x), where xxx is a random variable XXX. Below are the key derivatives:

  • First Derivative: f′(x)=1ln⁡(10)xf'(x) = \frac{1}{\ln(10) x}f′(x)=ln(10)x1​
  • Second Derivative: f′′(x)=−1ln⁡(10)x2f''(x) = -\frac{1}{\ln(10) x^2}f′′(x)=−ln(10)x21​
